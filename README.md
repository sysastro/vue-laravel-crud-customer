## Installation

####Setup Laravel .env
DB_CONNECTION=mysql

DB_HOST=127.0.0.1

DB_PORT=3306

DB_DATABASE=mnc

DB_USERNAME=root

DB_PASSWORD=

####Migration

php artisan migrate

####Composer

composer install

####NPM

npm install

####Build

npm run dev

####Run

php artisan serve

Laravel development server started: http://127.0.0.1:8000
